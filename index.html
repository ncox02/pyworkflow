The core idea

Before the details, here's the whole picture.

A modern Python developer uses a handful of tools — maybe four or five — that each play a specific role. Once you understand what each one is for, everything else clicks into place. The tools change. The workflow doesn't.

Five categories. Infinite combinations.

Every tool fits into one of these roles.

The landscape feels overwhelming because there are hundreds of tools. But they all do one of five things. Once you see the categories, you can evaluate any new tool instantly.

01

The Editor / IDE

Where you write and read code. An IDE (Integrated Development Environment) is a text editor with superpowers — it understands your code, highlights errors, helps you navigate, and runs your programs.

VS Code · PyCharm · Cursor · Zed
Learn about IDEs →
02

The AI Assistant

A model that helps you write, explain, debug, and refactor code. Can live inside your IDE or as a separate chat interface. Transforms what a solo developer can accomplish in a day.

Claude · GitHub Copilot · ChatGPT · JetBrains AI
Learn about AI tools →
03

The Environment

A self-contained Python installation for each project. Keeps your projects isolated so their dependencies don't conflict. Every professional Python project has one.

venv · conda · uv · poetry
Learn about environments →
04

Version Control

A system that tracks every change to your code, lets you go back in time, and enables multiple people to work on the same project without overwriting each other.

Git · GitHub · GitLab · Bitbucket
Learn about Git & GitHub →
05

Package Management

How you install, update, and manage the libraries your code depends on. Works alongside your virtual environment to keep everything reproducible and shareable.

pip · conda · uv · poetry
Learn about packages →
"The tools change.
The workflow doesn't."

Whether you're using VS Code or PyCharm, Copilot or Claude, pip or poetry — the underlying pattern is identical. You write code in an editor, manage your dependencies in an environment, and save and share your work with Git.

Learning the pattern means you can switch tools at any time without starting from scratch. That's the goal of this site.
